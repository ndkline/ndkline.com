Elements Web Conference

Developed website for Penn State's yearly web conference

Worked as a team to develop a web application for the yearly web conference hosted by PSU. The site was built using Drupal 7, the Conference Organizing Distribution (COD) module, and Zurb Foundation 5. As a full stack developer I contributed from custom module development to writing SASS for theming.

  • Client: The Pennsylvania State University
  • Date: June 13-15, 2016
  • Service: Full Stack Developer
