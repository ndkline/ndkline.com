CETC Kiosk

Developed a IT ticketing portal and system

Developed a responsive frontend application for the College of Education’s IT Department, the Carrara Education Technology (CETC), to handle ticket submissions. The applications main purpose was to be used in a kiosk outside the department however it’s flexible nature allows it to be used on any device easily. It displays a news feed and campus warnings in additions to its targeted goals.
